A.9 Git Commands

A.9.1 Setup and Configuration

Command Description
git config --global user.name "Name" Set username
git config --global user.email "email" Set email
git config --list Show configuration

A.9.2 Repository Operations

Command Description
git init Initialize repository
git clone URL Clone remote repository
git status Show current state
git log --oneline View commit history

A.9.3 Making Changes

Command Description
git add file Stage file
git add . Stage all changes
git commit -m "message" Commit staged changes
git diff Show unstaged changes
git diff --staged Show staged changes

A.9.4 Branching

Command Description
git branch List branches
git branch name Create branch
git checkout name Switch branch
git checkout -b name Create and switch
git merge name Merge branch
git branch -d name Delete branch

A.9.5 Remote Operations

Command Description
git remote -v Show remotes
git remote add origin URL Add remote
git push origin branch Push to remote
git pull origin branch Pull from remote
git fetch origin Fetch without merge

A.10 SLURM Commands

A.10.1 Job Submission

Command Description
sbatch script.sh Submit batch job
srun --pty bash Start interactive job

A.10.2 Job Management

Command Description
squeue -u $USER Show your jobs
squeue -p partition Show partition jobs
scancel JOBID Cancel job
scancel -u $USER Cancel all your jobs
scontrol show job JOBID Job details
sacct -j JOBID Job accounting

A.10.3 Information

Command Description
sinfo Partition information
sinfo -p partition Specific partition

A.10.4 Common SBATCH Directives

#SBATCH --job-name=name       # Job name
#SBATCH --partition=compute   # Partition
#SBATCH --account=PIRG        # Account
#SBATCH --time=HH:MM:SS       # Time limit
#SBATCH --nodes=1             # Number of nodes
#SBATCH --cpus-per-task=4     # CPUs per task
#SBATCH --mem=16G             # Memory
#SBATCH --gres=gpu:1          # GPUs
#SBATCH --output=out_%j.log   # Output file
#SBATCH --error=err_%j.log    # Error file

A.11 Module Commands

Command Description
module avail List available modules
module spider name Search for module
module load name Load module
module unload name Unload module
module list Show loaded modules
module purge Unload all modules

A.12 R Quick Reference

A.12.1 Basic Operations

# Assignment
x <- 5

# Vectors
v <- c(1, 2, 3, 4, 5)

# Sequences
1:10
seq(0, 1, by = 0.1)

# Basic stats
mean(v)
sd(v)
summary(v)

A.12.2 Data Frames

# Create
df <- data.frame(x = 1:3, y = c("a", "b", "c"))

# Access
df$x
df[1, ]
df[, "x"]

# Filter
df[df$x > 1, ]

# Read/Write
read.csv("file.csv")
write.csv(df, "file.csv", row.names = FALSE)

A.12.3 Getting Help

?function_name
help(function_name)
example(function_name)

A.14 Useful Shortcuts

A.14.1 Bash Keyboard Shortcuts

Shortcut Action
Ctrl+C Cancel current command
Ctrl+D End of input / logout
Ctrl+L Clear screen
Ctrl+A Move to start of line
Ctrl+E Move to end of line
Ctrl+R Search command history
Tab Autocomplete
↑ / ↓ Previous/next command

A.14.2 RStudio Keyboard Shortcuts

Shortcut Action
Ctrl+Enter Run current line
Ctrl+Shift+Enter Run current chunk
Ctrl+Shift+S Run entire script
Ctrl+Shift+C Comment/uncomment
Ctrl+Shift+M Insert pipe %>%
Alt+- Insert assignment <-
Ctrl+Shift+K Knit document
