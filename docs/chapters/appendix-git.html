B.13 Useful Aliases

Add these to your ~/.gitconfig for convenience:

[alias]
    st = status
    co = checkout
    br = branch
    ci = commit
    lg = log --oneline --graph --decorate
    last = log -1 HEAD
    unstage = reset HEAD --
    visual = !gitk
    aliases = config --get-regexp alias
